# AIDefenseX

## Overview
AIDefenseX is an AI-powered Intrusion Detection and Prevention System (IDPS) that combines Machine Learning, Wazuh, and Suricata to detect and prevent cyber attacks in real time.

## Features
- Real-time threat detection
- AI-based attack classification
- Wazuh SIEM integration
- Suricata IDS/IPS integration
- Malicious IP blocking
- Reduced false positives

## Technologies Used
- Python
- Wazuh
- Suricata
- Scikit-learn
- Nmap
- Linux

## Attack Detection
- Port Scanning
- Brute Force Attacks
- Malware Activity
- Suspicious Network Traffic

- Advanced Machine Learning Models
